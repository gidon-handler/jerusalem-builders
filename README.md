# Bnei Yerushalayim Construction — Company Website

A bilingual (English / Hebrew) marketing website for a Jerusalem-based building company, built with TanStack Start and Tailwind CSS v4.

## Features

- **Four service categories**: Interior Design, Exterior Construction, Safe Rooms (Mamad), Capsule Homes
- **Three demo projects per category** with images, descriptions, location, year and size
- **English ↔ Hebrew toggle** with full RTL layout support
- Beautiful dark design with Jerusalem amber/gold accent colours
- Animated hero with background imagery
- Stats bar, project cards with hover effects, contact form, sticky navbar

## Tech Stack

- [TanStack Start](https://tanstack.com/start) (React SSR framework)
- [TanStack Router](https://tanstack.com/router) (file-based routing)
- [Tailwind CSS v4](https://tailwindcss.com) (utility-first styling)
- [Heebo](https://fonts.google.com/specimen/Heebo) Google Font for Hebrew text
- Deployed on [Netlify](https://netlify.com)

## Running Locally

```bash
npm install
npm run dev
```

The dev server starts at http://localhost:3000.

For full Netlify feature emulation (functions, edge, etc.):

```bash
netlify dev --port 8889
```
